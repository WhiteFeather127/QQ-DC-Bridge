# 改动总结

## 一、Bug 修复 — `src/bridge/orchestrator.py`

### 1. 跳过翻译时输出 `None` + 丢失图片（核心 Bug）

**根因**  
`handle_qq_message` 中，当 `/distrans` 或 `should_skip` 触发跳过翻译时，`translated` 保持 `None`，但仍执行了：

```python
text = f"`{event.author_name}`: {translated}"  # → "`QQUser`: None"
segments_to_send = [text_segment(text)]           # 丢弃了图片等非文本段
```

**修复**  
重构条件分支，跳过翻译时走 `else` 分支 `[prefix] + converted`，保留图片。

```diff
- if self.translator and original_text:
-     if not skip_translation:
-         翻译...              # translated 被赋值
-     text = f"`{author}`: {translated}"   # 跳过时 → None
-     segments_to_send = [text_segment(text)]
- else:
-     segments_to_send = [prefix] + converted

+ skip_translation = False
+ if self.translator and original_text:
+     if self.translator.should_skip(original_text):
+         skip_translation = True
+     elif "/distrans" in original_text:
+         skip_translation = True
+
+ if self.translator and original_text and not skip_translation:
+     翻译...
+     segments_to_send = [text_segment(text)]
+ else:
+     # 跳过翻译 / 无文本 / 无翻译器 → 原文转发，保留图片
+     segments_to_send = [prefix] + converted
```

### 2. 翻译失败 / 译文与原文相同时仍输出 `None`

**根因**  
翻译 API 返回 `None`（失败）或译文规范化后与原文相同时，`translated` 被置为 `None`，但仍走了翻译格式输出路径。

**修复**  
在翻译路径内嵌套 `if translated is not None:` 判断，否则回退到 `else` 分支原文转发。

```diff
  if self.translator and original_text and not skip_translation:
      translated = await self.translator.translate(...)
      if translated is not None:
          if _normalize_text(translated) == _normalize_text(original_text):
              translated = None
+     if translated is not None:
          文本 = f"`{author}`: {translated}"
          segments_to_send = [text_segment(文本)]
+     else:
+         # 翻译失败或与原文相同 → 原文转发
+         segments_to_send = [prefix] + converted
```

### 3. Discord→QQ 方向缺少 `/distrans` 支持

**根因**  
`handle_discord_message` 中没有检查 `/distrans`。

**修复**  
增加了与 QQ 方向相同的 `skip_translation` 检测逻辑：

```python
skip_translation = False
if self.translator and original_text:
    if self.translator.should_skip(original_text):
        skip_translation = True
    elif "/distrans" in original_text:
        skip_translation = True

if self.translator and original_text and not skip_translation:
    # 翻译...
else:
    segments_to_send = [prefix] + converted
```

### 4. `/distrans` 命令泄露到目标平台

**修复**  
检测到 `/distrans` 后，在发送前从所有 `SEGMENT_TEXT` 段中剥离该字符串：

```python
has_distrans = "/distrans" in original_text
if has_distrans:
    for seg in converted:
        if seg.type == SEGMENT_TEXT:
            seg.data["text"] = seg.data.get("text", "").replace("/distrans", "").strip()
    _, original_text = self.translator.extract_text_segments(converted)
```

## 二、功能增强 — `src/bridge/translator.py`

### 5. `should_skip` 增加代码内容检测

添加了代码内容启发式检测，识别代码片段、堆栈跟踪、配置代码等，跳过翻译以节省 API 调用。

```diff
  def should_skip(self, text: str) -> bool:
      if re.fullmatch(r"https?://\S+", text):
          return True
-     if text.startswith("```") and text.endswith("```"):
-         return True
+     if self._is_code_content(text):
+         return True
      return False
+
+ def _is_code_content(self, text: str) -> bool:
+     """启发式检测消息是否为代码内容"""
+     # 代码块标记
+     if "```" in text:
+         return True
+     # 堆栈跟踪特征
+     if re.search(r'File\s+".*?",\s*line\s+\d+', text):
+         return True
+     if re.search(r'\s+at\s+[\w.]+\(.*?\)', text):
+         return True
+     # 逐行检查代码特征
+     code_line_re = re.compile(
+         r'^\s*('
+         r'(def|class|import|from|return|if|elif|else|for|while|try|except|finally|with|async|await|'
+         r'pass|break|continue|raise|yield|lambda|print|assert|del|global|nonlocal)\b'
+         r'|(public|private|protected|static|void|int|string|bool|float|double|char|var|val|fun|'
+         r'const|let|function|interface|type|enum|record)\b'
+         r'|(using|namespace|include|define|pragma|package|require|module\.exports)'
+         r'|(console\.|System\.|std::|fmt::|log\.|logger\.|print\()'
+         r'|@\w+'
+         r'|[\w.]+\(.*\)\s*\{'
+         r'|export\s+(default\s+)?(function|class|const|let|var|interface|type)'
+         r')\s*'
+     )
+     lines = text.split('\n')
+     non_empty = [l for l in lines if l.strip()]
+     if not non_empty:
+         return False
+     code_lines = sum(1 for l in non_empty if code_line_re.match(l))
+     if len(non_empty) > 1:
+         return code_lines / len(non_empty) > 0.3
+     return code_lines > 0
```
